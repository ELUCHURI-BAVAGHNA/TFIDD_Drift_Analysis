# TFIDD Drift Analysis – Extended Visualizations

This repository contains extended visualizations supporting the TFIDD drift detection study submitted for review.

## Contents

- **SEA** – SEA dataset figures across chunk sizes 300, 500, 800, 1000
- **Hyperplane** – RBF dataset visualizations
- **Agarwal** – Agarwal stream analysis figures
- **Electricity** – Real-world Electricity dataset
- **Forest_Covertype** – Forest Cover stream visuals
- **Gas_Sensor** – Annotated drift visualizations for Gas dataset

Each folder contains:
- Prequential accuracy
- KS / MI distribution shift plots
- Drift score plots
- Feature importance and trajectory plots
- Stability vs recovery plots
- Cross-chunk alarm analysis
